# AURORA Web Front-End

Projeto desenvolvido para a Global Solution da FIAP, unindo Front-End Design e Web Development em uma landing page sobre o sistema AURORA: uma proposta de gestão energetica autonoma para bases lunares e ambientes isolados.

## Sobre o Projeto

AURORA é um sistema conceitual voltado para monitoramento, priorização e previsão de autonomia energetica. A solução foi pensada para cenários extremos, como uma base lunar durante a noite de aproximadamente 14 dias terrestres, quando paineis solares deixam de gerar energia e cargas criticas precisam ser protegidas.

O site apresenta o problema, a tecnologia, os objetivos, o publico-alvo, os beneficios e a aplicação prática da solução. Tambem possui uma página dedicada a missão, com uma explicação mais detalhada do contexto e da proposta.

## Funcionalidades

- Landing page responsiva sobre o projeto AURORA.
- Pagina complementar `missao.html` com contexto expandido da missao.
- Slideshow no hero com navegacao manual, indicadores e progresso automatico.
- Tres temas de cor: Dia, Noite Lunar e Alerta.
- Formulario de contato com validação visual em JavaScript.
- Exibição dos dados enviados pelo formulário no console.
- Quiz dinâmico com 10 perguntas sobre a missão AURORA.
- Feedback visual no quiz para respostas certas e erradas.
- Resultado final e botão para reiniciar o quiz.

## Como Executar

1. Baixe ou clone o repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `Aurora-Web-Front/index.html` em um navegador.

Não é necessário instalar dependências, pois o projeto utiliza apenas HTML, CSS e JavaScript puro.
